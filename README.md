# Smart Parking System using ESP32

## Overview
This project detects vehicle presence in parking slots using ultrasonic sensors and displays available parking spaces in real time.

## Features
- Vehicle Detection
- Slot Availability Monitoring
- LED Status Indicators
- Real-Time Parking Information

## Components
- ESP32
- HC-SR04 Sensors
- LEDs
- Breadboard

## Working
Ultrasonic sensors detect vehicles parked in slots. The ESP32 calculates available spaces and indicates parking status through LEDs.
